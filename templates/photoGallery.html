KC Summer Slam Series T-Shirt Logo

For this project, we were asked to make a design that would be put onto a T-Shirt for our local high school tennis tournament series. For this design, I got my inspiration from the NHL Minnesota Stadium Series Logo and went to work trying to figure out how I could transform the elements of that logo into a design that fit the requirements of the project that look amazing on a white T-Shirt. In doing this, I made the main background color white so that it blended well with the shirt, and as we were required to use the logos of each of the hosting schools, I placed those in the location of the trees in the Minnesota logo.

Software Used: Adobe Photoshop CS6

Racecar 3 - Team Cars

Each of these cars were created for the Racecar 3 game I worked on. The way each of these cars are created in the game is that I have a class that takes in 3 colors, the main color, the secondary color, and the alt color, and the program then takes those colors and fills in the proper image of the slices of the racecar completely solid with the color suppose to cover that secion, and stack the images on top of each other so that a proper car is displayed to the user. These cars are based off of the 2017 McLaren Honda and takes a very heavy influence from the Mini Drivers web series car design.

Software Used: Pixlr Image Editor, Java Programming Language

Racecar 3 - Team Cars

Each of these track previews were created for the Racecar 3 game I worked on. Each preview is meant to give the user an idea about what the track will look like going into the upcoming race weekend so that they have an idea about the levels of downforce required to run the fastest times around the track would be as well as to get the user more immersion that each new race is different and gives them a visual appeal doing so. Each track is also divided into 3 main sectors colored differently on the map so the users know where the sector end points are. Each track shown was taken from the actual Formula 1 calender and would hence need to be modified if I were to ever even attempt at selling the game.

Software Used: Pixlr Image Editor

Racecar 3 - Billboards


Racecar 3 - Team Cars

These are all of the billboards that will be used in the background of the Racecar 3 game I worked on. The idea of these billboards is firstly to help show the user that the cars are moving along the race track as the curbs along the track do not fully convence the user. They are also there to give the user a bit of a laugh as a lot of the signs are just pure nonsense whether it be for a team attacking another team or its drivers or just a completely made up advertisement for a non sensical product. None of these billboards were ever meant to be taken seriously and I believe that the custom signs look better than what Formula 1 does having one ad repeated hundreads of times in a row on a section of the tracks walls. For the game, each of these billboards will be distorted slightly to help give the illusion that the track view is diagonal.

Software Used: Pixlr Image Editor
